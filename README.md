  Система фитнес клубов [5element.ua](https://5element.ua/) предоставляет различные услуги. В таблице service хранятся виду предоставляемых услуг.

  У каждого **тренера** есть список доступных **видов тренировок** и цена за час.

  **Клиент** регистрируется на сайте сети клубов, внося свое имя, фамилию,  электронную почту и номер телефона. По желанию клиент также может нанять тренера.

  Таблица **subscription** содержит информацию обо всех действующих абонементах клиентов.

  В системе клубов ведётся **история всех посещений** и занятий с тренерами.
